# Politica de Seguridad

## Versiones Soportadas
Actualmente solo se proporcionan parches de seguridad y soporte activo para la rama `main` (produccion).

## Como reportar una vulnerabilidad
Si encuentras un problema de seguridad critico, por favor no abras un Issue publico.
Envia un reporte directo al correo de los administradores del repositorio con los detalles del problema. El equipo lo evaluara y aplicara un parche a traves de una rama `hotfix` lo antes posible.
