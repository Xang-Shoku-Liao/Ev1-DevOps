# Guia de Contribucion

Para contribuir a este proyecto, por favor sigue las siguientes reglas:

1. Crea una nueva rama a partir de `develop` para tu trabajo (ejemplo: feature/nombre-funcionalidad).
2. Realiza tus cambios y asegúrate de que el código funcione correctamente.
3. Haz commits descriptivos siguiendo la convención del proyecto (feat, fix, docs).
4. Sube tu rama y abre un Pull Request hacia `develop`.
5. Espera la revisión y aprobación de un compañero antes de fusionar el código.
